# mf-test

#### 介绍

技术栈为vite + @module-federation/vite + vue3 + react19 + turborepo + pnpm，主应用apps/main，子应用apps/vue和apps/react，公共库为packages/mf-runtime-loader、packages/mf-shared、packages/mf-telemetry。

#### 软件架构

vite + @module-federation/vite + vue3 + react19 + turborepo + pnpm

#### 安装教程

1. 安装node（版本20.19.x）
2. 安装pnpm（版本10.26.0）

#### 使用说明

1. pnpm install
2. 本地启动：pnpm dev
